HAMMING CODE
1. Hamming Coding Kya Hoti Hai?
Hamming Code ek error detection aur correction technique hai jo Richard Hamming ne 1950 mein banai thi.
Iska main purpose hai:
Data transmission mein agar 1-bit ka error ho jaye to usko detect bhi karna aur correct bhi karna.
2-bit errors ko detect to kar leti hai, lekin correct nahi karti.
Ye method digital communication systems mein use hoti hai, jaise:
Computer memory (RAM ECC)
Satellite communication
Modems / Networking
